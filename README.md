# MERN E-Commerce

A minimal MERN e-commerce website.

## Prompts used
- "Build a complete MERN stack E-Commerce website... (full requirements as provided by the user)."
- "Create project structure, backend (server.js, routes, controllers, models), frontend (components, pages, App.js, context), and TECH_DOC.md."

## Quickstart
### Backend
1. Copy `.env.example` to `.env` and set `MONGODB_URI`.
2. Install dependencies and run:
   ```bash
   npm i
   npm run seed
   npm run dev
   ```

### Frontend
1. Optionally set `VITE_API_BASE=http://localhost:5000/api` in `.env`.
2. Install and run:
   ```bash
   npm i
   npm run dev
   ```

Open http://localhost:5173

See `TECH_DOC.md` for more details.
